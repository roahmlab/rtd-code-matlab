Utility Types and Enumerations

class rtd.util.types.LogLevel

Bases: uint8

Enumeration type for logging levels.

Note

Log levels GENERAL and OFF have the same level so that GENERAL output will always show. These levels are roughly selected based on log4j and https://www.section.io/engineering-education/how-to-choose-levels-of-logging/.
